# Category & Product REST API (Spring Boot + JWT)

This project is a **RESTful API** built using **Spring Boot** that allows CRUD operations for **Categories** and **Products**, secured using **JWT (JSON Web Token) Authentication**.

## 🔐 Features

- ✅ **JWT-based Authentication & Authorization**
- 📦 **CRUD APIs** for:
  - Categories
  - Products
- 🔗 **Entity Relationships**:
  - A product belongs to a category
- 🧾 Well-structured DTOs, Services, and Repositories
- 📁 Layered architecture (Controller, Service, Repository)

## 🛠️ Technologies Used

- Java 17+
- Spring Boot
- Spring Security
- Spring Data JPA (Hibernate)
- JWT (jjwt or similar)
- MySQL / H2 (for DB)
- Maven
